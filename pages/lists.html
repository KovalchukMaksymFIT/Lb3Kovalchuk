Використання тегів для роботи з списками:

ul - Маркований список (li - елементи списку)

  • Алгебра
  • Геометрія
  • Фізика

ol - Нумерований список (li - елементи списку)

  1. Італія
  2. Британія
  3. Україна

dl - Список визначень; dt - Термін; dd - Опис терміну

HTML
Це не мова програмування, це мова розмітки, яка каже вашому браузеру, як відображати вміст веб-сторінки, яку ви переглядаєте.
CSS
Каскадна таблиця стилів, що використовується для опису зовнішнього вигляду сторінок.
